Automation Training

Where can I get CREOSON training?

Since CREOSON was developed as a generic interface for multiple langauges to communicate with, you are free to choose and use whichever language you want to interact with CREO via CREOSON.

As a result, CREOSON was designed to NOT require much training to be able to use it. However, the language you choose to use must be able to make JSON requests over the HTTP (web) protocol.

CREOSON was designed for you to "paint with a broad brush" and do things quickly, but success will always be dependent on your skills with the programming/scripting language you choose in combination with how well you have plannd your automation workflow with CREO Parametric.

Where can I get CREO Automation training?

One of the most important success factors in CREO Automation is to NOT try to replicate what a user does (meaning the click-by-click process) for automating models and/or drawings.

CREO Automation training is an excellent way to build the practical skills necessary to make CREO do amazing things in a highly repeatable and predicable way.

Simplified Logic offers CREO Automation Training -- using Nitro-CELL. Since Nitro-CELL uses the core of CREOSON for automation, the training is very aligned and useful for anyone wanting to use CREOSON. Most of the automation class is focused on how to properly model for change, the flow of data and properly execute automation in CREO.

Please contact us to discuss your needs. Customized training is available.
